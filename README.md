# EngenhariaUnificadaI

Para a disciplina de Engenharia Unificada I fizemos um protótipo de uma esteira que separa os materiais recicláveis em sua devida categoria (vidro, plástico e metal), assim facilitando o processo de reciclagem. 
O protótipo foi constituído de uma esteira automática, sensores que serão responsáveis pela identificação do tipo do material e um sistema de separação, todos controlados por arduíno. O objetivo é que o protótipo, quando concluído, seja capaz de identificar o tipo de material reciclável colocado na esteira e possa separá-lo corretamente
